🩺 AI Health Assistant Chatbot

An AI-powered health chatbot built using Python, Google Gemini API, and Gradio.
It provides simple, safe, and informative responses to general health-related questions.

🚀 Features
💬 Conversational AI chatbot
⚡ Fast responses using Gemini (gemini-2.5-flash)
🛡️ Safety filtering for sensitive queries
⚙️ Quick replies for common messages (hello, thanks, etc.)
🌐 Interactive UI using Gradio
📌 Simple and user-friendly design
🛠️ Tech Stack
Python
Google Generative AI (Gemini API)
Gradio
NLP (Natural Language Processing)

⚙️ Installation & Setup
1️⃣ Clone the Repository
git clone https://github.com/your-username/health-chatbot.git
cd health-chatbot
2️⃣ Install Dependencies
pip install google-generativeai gradio
3️⃣ Add Your API Key

Open the code and replace:
genai.configure(api_key="YOUR_API_KEY")
▶️ Running the Project
Option 1: Run Python Script
python app.py
Option 2: Run Jupyter Notebook
jupyter notebook

Then open Health_Chatbot.ipynb and run all cells.

💡 Example Queries
What causes a sore throat?
How to reduce fever naturally?
What are symptoms of flu?
How can I improve my sleep?
🛡️ Safety Disclaimer

⚠️ This chatbot provides general health information only.
It does NOT provide medical diagnosis or treatment.

👉 Always consult a qualified healthcare professional for serious medical issues.
